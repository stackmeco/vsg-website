### **Key Physics Changes in this Code:**

1.  **`grayscale(100%)`**: We strip the color from the image so it doesn't clash with your **International Orange**. It forces the image to become part of the "Tungsten" environment.
2.  **`opacity: 0.2`**: This is the "Ghost" level. It is visible enough to provide depth and scale (the "Grid"), but transparent enough that the text reading "Engineering verifiable systems..." pops aggressively.
3.  **`rounded-[1px]`**: Notice the small details like the status dot. I changed it from `rounded-full` (circle) to `rounded-[1px]` (square-ish). This aligns with the new "Machined" aesthetic.
