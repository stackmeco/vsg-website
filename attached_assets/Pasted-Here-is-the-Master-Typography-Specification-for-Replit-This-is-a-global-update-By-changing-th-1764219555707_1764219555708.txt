**TYPOGRAPHY SYSTEM UPGRADE: MANROPE INTEGRATION**

We are switching the primary UI font from **Inter** to **Manrope**.
**Reason:** Manrope is geometric and semi-condensed. It feels more "engineered," "machined," and "proprietary" than the default Inter. It aligns better with the Space Grotesk headers.

Execute these 3 steps to apply this change globally.

### STEP 1: UPDATE ASSET LOADING (`client/index.html`)
**Action:** Replace the Google Fonts `<link>` tags in the `<head>` with this updated block that includes Manrope.

```html
<link rel="preconnect" href="[https://fonts.googleapis.com](https://fonts.googleapis.com)">
<link rel="preconnect" href="[https://fonts.gstatic.com](https://fonts.gstatic.com)" crossorigin>
<link href="[https://fonts.googleapis.com/css2?family=IBM+Plex+Mono:wght@400;500;600&family=Manrope:wght@300;400;500;600;700&family=Space+Grotesk:wght@400;500;600;700&display=swap](https://fonts.googleapis.com/css2?family=IBM+Plex+Mono:wght@400;500;600&family=Manrope:wght@300;400;500;600;700&family=Space+Grotesk:wght@400;500;600;700&display=swap)" rel="stylesheet">
````

### STEP 2: UPDATE TAILWIND CONFIG (`tailwind.config.ts`)

**Action:** Update the `fontFamily` object in `theme.extend`. Map the default `sans` stack to Manrope.

```typescript
// In tailwind.config.ts
fontFamily: {
  // Replaces 'Inter' with 'Manrope' as the default sans
  sans: ['Manrope', 'sans-serif'], 
  // Keep Space Grotesk for headings
  heading: ['Space Grotesk', 'sans-serif'],
  // Keep IBM Plex Mono for data
  mono: ['IBM Plex Mono', 'monospace'],
},
```

### STEP 3: GLOBAL OPTICAL ADJUSTMENT (`client/src/index.css`)

**Action:** Manrope is slightly wider and friendlier than Inter. To maintain the "Control Room" density, we must tighten the letter spacing globally.
Update the `@layer base` block:

```css
@layer base {
  body {
    @apply bg-background text-foreground font-sans antialiased;
    /* TIGHTEN MANROPE: This creates the "Machined" look */
    letter-spacing: -0.015em; 
  }

  /* Ensure headings remain tight and distinct */
  h1, h2, h3, h4, h5, h6 {
    @apply font-heading tracking-tight;
  }
  
  /* Enforce tabular numbers for data */
  .tabular-nums {
    font-feature-settings: "tnum";
    font-variant-numeric: tabular-nums;
  }
}
```
