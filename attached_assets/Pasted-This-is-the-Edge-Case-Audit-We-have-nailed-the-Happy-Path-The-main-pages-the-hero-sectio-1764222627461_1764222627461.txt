### **1. The Command Palette "Empty State"**

  * **The Issue:** If a user searches for "Crypto" and finds nothing, the default Shadcn component likely says: *"No results found."*
  * **The Fix:** It should say: **"NO MATCHING RECORDS."** or **"SIGNAL LOST."**
  * **Why:** It maintains the "Database/Terminal" framing even in failure.

### **2. The Contact Form "Success" Message**

  * **The Issue:** Currently, it says *"Message received. We'll get back to you..."* .
  * **The Fix:** It should look like a ticket creation.
      * **Headline:** **"TRANSMISSION LOGGED."**
      * **Body:** **"Your query has been queued for internal review. Reference ID: [Random Hex Code]."**
  * **Why:** It feels like you interacted with a machine, not a support desk.

### **3. The "Legal" Page Typography**

  * **The Issue:** Privacy Policy and Terms usually look like generic text blocks.
  * **The Fix:** Force the **Headers** of these pages to be `font-mono text-xs uppercase`.
  * **Why:** It makes the legal documents look like **"System Protocols"** or **"Classified Docs"** rather than standard web pages.

### **4. The "Favicon" (The Browser Tab)**

  * **The Issue:** You likely still have the default Vite logo in the browser tab.
  * **The Fix:** We will generate a code-based SVG favicon (International Orange Square) so you don't even need to open Photoshop.
